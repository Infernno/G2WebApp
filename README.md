# G2WebApp
G2 - небольшой (и недоделанный) клон сайта Пикабу на NancyFx + Linq2Db

# Что есть
- [x] Регистрация и авторизация
- [x] Лента постов
- [x] Создание постов
- [x] Комментарии
- [x] Голосование (как для постов, так и для комментариев)
- [x] Админ-панель с модерацией постов и комментариев
